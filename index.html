Prework Study Guide

HTML

  • The head element contains information about the webpage.
    • title - defines the title of the webpage (important for SEO)
    • style - defines the CSS styles associated with a website
    • base - defines the base URL for a webpage
    • link - connects an external resource to the HTML document
    • script - embeds client-side scripts in the HTML document
    • meta - defines metadata such as character sets, description, keywords, author, and viewport
      • charset - sets our character encoding
      • http-equiv - sets on HTTP header for content
      • viewport - sets the default view for a webpage
  • The body element represents the visible content shown to the user.
    • h1-h5 - represents the level of heading given to a text block
    • p - represents a paragraph or a block of text
    • ul - represents unordered list
    • ol - represents ordered list
    • li - represents list items
    • img - contains information about images that are displayed
      • src - defines the location of the image
    • br - creates a line of empty space ot a line break
    • a - (anchor) creates links to the same webpage or other webpages
      • href - points to the URL for the link

CSS

  • Cascading Style Sheets - programming language used by developers to define how webpage content is presented to users. (color, visual layout)
  • There are three different kinds of CSS
    • inline CSS - style an element directly in the HTML file (only applies to that element)
    • internal CSS - contains CSS rules for the webpage in the head section of the HTML file (only applies to that page)
    • external CSS - separate file for all CSS rules
  • declarations contain two components: CCS property and property value
  • margin - how much space around the outside
  • padding - how much space around the content inside an element
  • class attribute - share a CSS rule to any element by assigning the rule to a class attribute with a class selector

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript - high-level dynamic programming language that allows users to interact with websites
  • control flow - order in which a computer executes code in a script
  • array - single variable that is used to hold a group of data
  • for loop - uses the predictable pattern of indices to preform a task on all the items in the array by allowing a single code block to be executed over and over - contains three important statements:
    • the first statement determines the starting point for the loop ( var x = 0; )
    • the second statement is the condition. As long as the number in the variable(x) is less than the length of the array, the loop will keep running ( x < shapes.length; )
    • the final statement allows the array to iterate over each item ( x++ )
  • function - a set of instructions that tells the computer how to perform a certain task - you have to call it by name to execute ( function helloWorld() )
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
